# Next.js 시작하기

이 포스트는 Next.js를 시작하는 방법에 대한 더미 콘텐츠입니다.

## 개요

Next.js는 React 기반 프레임워크로 정적 생성과 서버 렌더링을 모두 지원합니다.

## 주요 기능

- App Router
- Static Site Generation
- 파일 기반 라우팅

### SSG란?

Static Site Generation의 약자입니다.

### 왜 쓰나?

콘텐츠 중심 사이트에서 빠른 로딩과 간단한 배포에 유리합니다.
