# Next.js 시작하기

이 포스트는 Next.js를 시작하는 방법에 대한 내용을 담고 있습니다.

## 1. 개요
Next.js는 리액트 프레임워크입니다.

## 2. 주요 기능
서버 사이드 렌더링과 정적 생성을 지원합니다.

### 2.1 SSG란?
Static Site Generation의 약자입니다.

### 2.2 SSR이란?
Server Side Rendering의 약자입니다.

## 3. 결론
Next.js는 정말 강력합니다.
